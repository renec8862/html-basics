HTML Basics

The "frontend" of a webpage consists of:

  • HTML = HyperText Markup Language
    • content
  • CSS = Cascading Style Sheet
    • style
  • JS = Javascript
    • interactivity

Vocabulary & Syntax

  • elements have the following syntax: attribute="value">content

Block vs. Inline

  • block-level elements always start on a new line and takes up the full width available
  • inline-elements do not start on a new line and only takes up as much width as necessary
